# CRM SaaS Multi-tenant

A modern CRM SaaS platform with a multi-tenant architecture, featuring a luxury dark theme dashboard, leads pipeline, finance management, and subscription control.

## Project Structure

- `frontend/`: React + Vite application with custom luxury styling.
- `backend/`: FastAPI application using Excel for efficient data storage.

## Features

- **Multi-tenant Architecture**: Separate environments for different businesses.
- **Sales Pipeline**: Manage leads through customizable stages.
- **Finance Module**: Track income, expenses, and automated recurring billing.
- **Product Catalog**: Manage items and services with Excel import/export capability.
- **Subscription Management**: Automated contracts and subscription statuses.

## Tech Stack

- **Frontend**: React, Lucide Icons, Recharts, CSS Variables.
- **Backend**: FastAPI, Pydantic, Pandas, Openpyxl.
